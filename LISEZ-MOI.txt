LE PETIT BOOTH — CRM interne (V4 · 100% Supabase)
=================================================

L'application enregistre TOUT dans votre base Supabase (le "carnet central").
Résultat : où que vous ouvriez l'appli (téléphone, autre téléphone, ordinateur),
il suffit d'entrer identifiant + mot de passe pour retrouver toutes vos données.
Il n'y a PLUS de sauvegarde locale : Supabase est la seule base.

CE QU'IL FAUT FAIRE UNE SEULE FOIS (voir GUIDE-SUPABASE.txt)
  1. Créer un projet Supabase (gratuit).
  2. Exécuter le fichier « supabase-setup.sql » (SQL Editor de Supabase).
  3. Récupérer 2 valeurs : Project URL + clé "anon/publishable".
  4. Coller ces 2 valeurs en haut du fichier lepetitbooth-crm.html :
        const SUPABASE_URL      = 'https://VOTRE-REF.supabase.co';
        const SUPABASE_ANON_KEY = 'eyJ...';   // ou sb_publishable_...
  5. Mettre le fichier en ligne (Netlify Drop ou Vercel) pour l'ouvrir partout.

CONNEXION
  Identifiant : KevinObscur
  Mot de passe : celui défini dans supabase-setup.sql (par défaut KevinObscur974 ;
  vous pouvez le changer à la ligne _admin_pw du script SQL).

VÉRIFICATION
  En bas de l'écran de connexion :
   • « ● Base connectée » (vert)  → tout est bon.
   • message orange                → il indique précisément quoi corriger
     (clés manquantes, fichier non hébergé/hors-ligne, ou SQL non exécuté).

LIENS PRIVÉS (lecture seule, synchronisés en temps réel)

  Prestataire : .../lepetitbooth-crm.html#/prestataire/xxxxxxxx

IMPORTANT

  pour fonctionner. Tant que ce n'est pas fait, l'écran de connexion l'indique

  synchronise entre tous vos appareils, exactement comme les autres applis.
